> **Portfolio**
> A modern personal portfolio site built with React and Vite that showcases projects, skills and experience.
>
> **Key features**
>
> * Built using React for interactive UI and Vite for fast build & hot-reload.
> * ESLint configured for code quality.
> * Minimal setup with clear folder structure (`public`, `src`, etc.).
> * Deployment ready (for example via Vercel, as indicated by the linked URL).
>
> **Purpose**
> This site is intended to represent the developer’s (your) identity online — making it easy for recruiters, collaborators or clients to view your work, get contact info and assess your skills at a glance.
>
> **Usage**
>
> * Clone the repo
> * `npm install` to install dependencies
> * `npm run dev` to start the local dev server with hot-reload via Vite
> * `npm run build` to produce a production version
>
> **Tech stack**
>
> * JavaScript (React)
> * Vite as bundler/build tool
> * ESLint for linting
> * HTML/CSS for the layout & styling
>
> **Live Demo**
> A live version is available at: [https://nabh-yadav-portfolio.vercel.app](https://nabh-yadav-portfolio.vercel.app)
